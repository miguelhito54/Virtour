# Virtour-U

Virtour-U es un pequeño proyecto desarrollado por un grupo de 3 estudiantes de la Universidad Cooperativa de Colombia con el objetivo de diseñar una página web en la cual los estudiantes y aspirantes puedan explorar el campus Medellín-Envigado de la UCC de manera remota, enfocándose principalmente en la sede ubicada en el centro de Medellín, con el fin de facilitar el acercamiento a la planta física o su infraestructura.
